### Installation

Requires [Node.js](https://nodejs.org/) v16 to run.
Requires [YARN](https://yarnpkg.com/) v1.22.0 to run.

##### Frontend

With yarn

```sh
$ cd rule-of-thumb-project
$ yarn
$ yarn dev
```

With npm

```sh
$ cd rule-of-thumb-project
$ npm install
$ npm run dev
```
